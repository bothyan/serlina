## Usage

serlina-serve will map all pages in `/page` to route (files as router).

```
npm i react react-dom serlina-serve --save
```

```json
{
  "scripts": {
    "dev": "serlina-serve dev",
    "start": "serlina-serve start"
  }
}
```
